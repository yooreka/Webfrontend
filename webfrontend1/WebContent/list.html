  • Oracle

JAVA
클라이언트 프로그램이나 서버용 프로그램을 만들기 위한 프로그래밍 언어
DBMS
데이터를 효율적으로 저장하고 관리하기 위한 프로그램으로 Oracle, MySQL, MongoDB 등이 있음
Web Front End
웹 브라우저에 출력을 하기 위한 기술로 HTML, CSS, JavaScript 가 표준기술
Java Web Programming
Java를 이용해서 Web Server를 만드는 기술로 내장 객체, Servlet, JSP, Filter, Listener, EL, JSTL 등의 기술
Java Framework
Java로 프로그램을 구현할 때 도와주는 프로그램으로 Spring, MyBatis, Hibernate가 대표적인 프레임워크
Android
Java 나 Kotlin을 이용해서 안드로이드 운영체제에서 동작하는 앱을 개발
iOS
swift를 이용해서 애플 스마트 기기에서 동작하는 애플리케이션을 개발
